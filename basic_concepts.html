BCC语言基本概念

深入理解BCC语言的核心概念和特性

1. 语言特点

  • 简单易学：语法简洁，类似Python
  • 强类型：变量类型在运行时确定
  • 代码块特性：支持函数式编程风格
  • 模块化：支持模块导入和复用

2. 数据类型

2.1 基本类型

  • 数值类型：整数(Integer)和浮点数(Float)
  • 字符串(String)：使用双引号表示
  • 布尔值(Boolean)：true和false
  • 空值(Null)

2.2 复合类型

  • 数组(Array)：有序集合
  • 代码块(Codeblock)：可执行的代码片段
  • 对象(Object)：键值对集合

3. 变量

3.1 变量声明

BCC使用动态类型系统，变量无需显式声明类型：

name = "Alice"    // 字符串
age = 25         // 整数
height = 1.75    // 浮点数
isStudent = true // 布尔值

3.2 变量作用域

  • 全局作用域：在最外层定义的变量
  • 函数作用域：在函数内定义的变量
  • 代码块作用域：在代码块内定义的变量

4. 运算符

4.1 算术运算符

a + b    // 加法
a - b    // 减法
a * b    // 乘法
a / b    // 除法
a % b    // 取余

4.2 比较运算符

a == b   // 等于
a != b   // 不等于
a < b    // 小于
a <= b   // 小于等于
a > b    // 大于
a >= b   // 大于等于

4.3 逻辑运算符

a && b   // 逻辑与
a || b   // 逻辑或
!a       // 逻辑非

5. 控制流

5.1 条件语句

if(condition) {
    // 代码块
} else if(condition2) {
    // 代码块
} else {
    // 代码块
}

5.2 循环语句

// for循环
for(i = 0, i < 10, i = i + 1) {
    // 代码块
}

// while循环
while(condition) {
    // 代码块
}

6. 函数

6.1 函数定义

def public add(a, b) {
    return a + b
}

def private helper() {
    // 私有辅助函数
}

6.2 函数参数

  • 位置参数：按顺序传递的参数
  • 类型参数：指定参数类型的参数
  • 代码块参数：接受代码块的参数

7. 代码块

7.1 代码块定义

block = {
    println("This is a code block")
}

// 带参数的代码块
block = { x ->
    println(x)
}

7.2 代码块使用

// 定义高阶函数
def public repeat(times, block: BCC.Codeblock) {
    for(i = 0, i < times, i = i + 1) {
        nsreturn block
    }
}

// 使用代码块
repeat(3) {
    println("Hello!")
}
提示： 代码块是BCC的一个重要特性，它使得函数式编程风格成为可能。

8. 模块系统

8.1 模块导入

import "math"
import "string" as str

8.2 模块导出

// 在模块中导出函数
def public exportedFunction() {
    // 实现
}

// 私有函数不会被导出
def private internalFunction() {
    // 实现
}
注意事项：
  • 所有函数必须声明访问修饰符(public/private)
  • 代码块参数必须指定类型为BCC.Codeblock
  • 模块路径使用双引号
