BCC语言快速入门

15分钟快速上手BCC编程

1. 第一个BCC程序

让我们从一个简单的"Hello, World!"程序开始：

// hello.bcs
printnln("Hello, World!")
printnln("Welcome to BCC!")
提示： BCC源代码文件使用 .bcs 扩展名。

2. 基本语法

2.1 变量和赋值

// 变量赋值
name = "BCC"
age = 25
price = 99.9

// 打印变量
printnln("Name: " + name)

2.2 条件语句

// if 语句
score = 85
if(score > 60) T {
    printnln("及格")
}

2.3 循环

// for 循环
for(i = 0, i < 5, i = i + 1) T {
    printnln(i)
}

// while 循环
x = 0
while(x < 3) T {
    printnln(x)
    x = x + 1
}

3. 函数定义

// 定义公共函数
def public add(a, b) T {
    return a + b
}

// 使用函数
result = add(5, 3)
printnln(result)
说明： BCC中的函数必须指定 public 或 private 访问修饰符。

4. 代码块

// 定义接受代码块的函数
def public repeat(times, block: BCC.Codeblock) T {
    for(i = 0, i < times, i = i + 1) T {
        nsreturn block
    }
}

// 使用代码块
repeat(3, { -> printnln("Hello!") })

5. 下一步

现在您已经了解了BCC的基础知识，您可以：

  • 条件语句的条件必须用括号 () 包围
