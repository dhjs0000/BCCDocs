BCC语言API参考

完整的BCC语言API文档

1. 核心函数

1.1 输入输出

函数名 参数 返回值 描述
printnln(value) value: 任意类型 null 打印值并换行
input() 无 string 从标准输入读取一行文本

1.2 类型操作

函数名 参数 返回值 描述
typeof(value) value: 任意类型 string 返回值的类型名称
len(value) value: string/array number 返回字符串或数组的长度

2. 运算符

2.1 算术运算符

运算符 用法 描述
+ a + b 加法或字符串连接
- a - b 减法
* a * b 乘法
/ a / b 除法
% a % b 取余

3. 标准库模块

3.1 math模块

函数名 参数 返回值 描述
math.abs(n) n: number number 返回绝对值
math.max(a, b) a: number, b: number number 返回两个数中的较大值
math.min(a, b) a: number, b: number number 返回两个数中的较小值

3.2 string模块

函数名 参数 返回值 描述
string.upper(str) str: string string 转换为大写
string.lower(str) str: string string 转换为小写
string.trim(str) str: string string 去除两端空白

4. 错误处理

4.1 错误类型

错误类型 描述 示例场景
SyntaxError 语法错误 代码格式不正确
TypeError 类型错误 操作类型不匹配
ReferenceError 引用错误 访问未定义变量
ImportError 导入错误 模块导入失败
说明：
  • 所有内置函数都是全局可用的
  • 模块函数需要先导入模块才能使用
  • 运算符优先级遵循常规编程语言规则
注意：
  • 除数不能为0
  • 数组索引必须是非负整数
  • 函数调用参数数量必须匹配
