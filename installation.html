BCC语言安装指南

在不同平台上安装和配置BCC开发环境

1. 系统要求

  • Python 3.8 或更高版本
  • pip 包管理器
  • Git（可选，用于获取最新源码）

2. 通过pip安装

2.1 Windows

pip install bcc-lang

2.2 Linux/macOS

pip3 install bcc-lang
提示： 如果安装过程中遇到权限问题，可以尝试使用 sudo pip3 install bcc-lang

3. 从源码安装

git clone https://github.com/yourusername/bcc-lang.git
cd bcc-lang
pip install -e .

4. 验证安装

安装完成后，打开终端输入以下命令验证安装：

bcc --version

如果安装成功，将显示BCC的版本信息。

5. IDE支持

5.1 Visual Studio Code

我们推荐使用VS Code作为BCC的开发环境。您可以安装以下扩展：

  • BCC Language Support
  • BCC Debugger

5.2 其他编辑器

目前我们也支持：

  • Sublime Text
  • Atom
  • PyCharm

6. 配置开发环境

6.1 环境变量

安装完成后，BCC解释器会自动添加到系统PATH中。如果需要手动配置，请添加以下路径：

# Windows
%USERPROFILE%\AppData\Local\Programs\Python\Python3x\Scripts

# Linux/macOS
/usr/local/bin

6.2 编辑器配置

推荐的VS Code设置：

{
    "bcc.interpreter.path": "bcc",
    "bcc.formatting.enable": true,
    "bcc.linting.enable": true
}
常见问题：
  • 如果遇到"命令未找到"错误，请检查PATH环境变量
  • 如果安装失败，请确保您有最新版本的pip
  • Windows用户可能需要以管理员身份运行命令提示符

7. 下一步

安装完成后，您可以：

  • 阅读快速入门指南开始编写BCC代码
  • 查看示例代码了解BCC的使用方法
  • 配置VS Code扩展获得更好的开发体验
